# Speedooo – Speed Reading App

A minimal web app that displays text one word at a time at a chosen speed (WPM). Built with Next.js for fast, focused reading.

## Features

* Paste any paragraph
* Adjustable WPM (words per minute)
* Play / Pause / Restart
* Clean, distraction-free UI
* Responsive design

## Tech Stack

* Next.js
* React
* Tailwind CSS

## How It Works

The app splits input text into words and displays them sequentially based on selected WPM using a timed interval.

## Getting Started

```bash
git clone https://github.com/your-username/wordflow.git
cd speedooo
npm install
npm run dev
```

## Usage

1. Paste your text
2. Set your reading speed (WPM)
3. Click play and start reading
